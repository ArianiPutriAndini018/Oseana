# OSEANA

**Oseana** adalah aplikasi edukasi biota laut Indonesia berbasis **pulau, QR Code, quiz, gamifikasi, dan Sea Passport**. Aplikasi ini dirancang untuk mendukung **SDG 14: Life Below Water / Ekosistem Laut** melalui media pembelajaran mobile yang interaktif, visual, dan mudah digunakan.

## Deskripsi Singkat

Oseana merupakan aplikasi mobile edukatif yang membantu pengguna mengenal biota laut Indonesia melalui konsep petualangan pulau. Setiap pulau menjadi checkpoint pembelajaran yang berisi maksimal 3 biota, biodata singkat, fun fact, link video edukatif, quiz, dan pesan konservasi.

Melalui aplikasi ini, pengguna tidak hanya membaca informasi, tetapi juga menyelesaikan misi belajar, mengerjakan quiz, mengumpulkan XP, membuka badge, dan memperoleh stamp pulau pada Sea Passport.

## 1. Latar Belakang Masalah

Edukasi biota laut di aquarium, museum, pameran, maupun kegiatan field trip masih sering bersifat pasif karena informasi biasanya hanya disampaikan melalui label, poster, atau penjelasan singkat. Kondisi tersebut membuat pengunjung belum tentu memahami habitat, ancaman, keunikan, dan pesan konservasi dari biota laut yang dipelajari.

Oseana dibuat sebagai media edukasi mobile yang menjadikan pembelajaran biota laut lebih aktif, visual, dan terukur. Aplikasi ini mengenalkan biota laut Indonesia melalui petualangan pulau, QR Code, quiz, progres belajar, dan Sea Passport agar pengguna tidak hanya membaca informasi, tetapi juga menyelesaikan misi belajar.

### Data Pendukung

* **[S1] UN SDG Report 2025**
  Menunjukkan bahwa baru sekitar **8,4% wilayah laut dunia** yang masuk kawasan perlindungan laut, sedangkan target global pada 2030 adalah **30%**. Data ini menunjukkan bahwa edukasi dan partisipasi masyarakat terhadap perlindungan laut masih perlu ditingkatkan.

* **[S7] Pemetaan Nasional Karang dan Lamun 2025**
  Mencatat bahwa Indonesia memiliki sekitar **2,3 juta hektare terumbu karang** dan sekitar **660 ribu hektare padang lamun**. Data ini menunjukkan bahwa Indonesia memiliki kekayaan ekosistem laut yang besar dan relevan untuk dikenalkan melalui media edukasi digital.

* **[S3] UNESCO 2026**
  Menampilkan contoh edukasi laut modern melalui pameran interaktif berbasis teknologi digital atau immersive tools. Hal ini menunjukkan bahwa edukasi laut dapat dikembangkan dengan pendekatan visual dan interaktif agar lebih mudah dipahami.

* **[S5] Frontiers in Education 2025**
  Menunjukkan bahwa pembelajaran berbasis gamifikasi memiliki hubungan positif dengan kesadaran ekologis siswa. Data ini mendukung penggunaan level, quiz, XP, badge, dan Sea Passport pada aplikasi Oseana.

* **[S6] UNEP 2026**
  Mencatat bahwa sekitar **19–23 juta ton sampah plastik** bocor ke ekosistem perairan setiap tahun. Data ini mendukung pentingnya pesan konservasi, khususnya ajakan mengurangi plastik sekali pakai dan menjaga ekosistem laut.

## 3. Tujuan dan Kesesuaian SDG 14

Oseana bertujuan mengenalkan biota laut Indonesia melalui pendekatan digital yang interaktif dan mudah dipahami. Aplikasi ini mendukung **SDG 14: Life Below Water** dengan membantu pengguna memahami habitat, ancaman, fun fact, dan pesan konservasi biota laut.

Tujuan utama aplikasi ini adalah:

1. Mengenalkan biota laut Indonesia berdasarkan pulau atau wilayah.
2. Meningkatkan literasi pengguna terhadap ekosistem laut dan konservasi.
3. Membuat pembelajaran biota laut lebih menarik melalui QR Code, quiz, gamifikasi, dan Sea Passport.
4. Mendukung kegiatan edukasi di aquarium, museum, pameran, field trip, maupun belajar mandiri.

## 7. Alur Kerja, QR, dan Gamifikasi

Alur pengguna:

```text
Buka aplikasi → pilih mode belajar → pilih pulau / scan QR → detail pulau & biota → quiz → result quiz → stamp terbuka → Sea Passport
```

| Tahap | Aksi Pengguna                                                  | Output Sistem                                                                       |
| ----- | -------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| 1     | Membuka aplikasi dan memilih mode belajar.                     | Sistem menampilkan pilihan Aquarium Mode dan Explore Mode.                          |
| 2     | Memilih pulau pada Ocean Journey Map atau scan QR panel pulau. | Sistem membuka detail pulau sesuai ID QR/level.                                     |
| 3     | Membaca materi pulau dan maksimal 3 biodata biota.             | Sistem menampilkan habitat, ancaman, fun fact, media/video, dan pesan konservasi.   |
| 4     | Menjawab quiz 3 soal.                                          | Sistem menghitung skor dan menentukan jumlah bintang.                               |
| 5     | Quiz lulus.                                                    | XP bertambah, stamp pulau masuk ke Sea Passport, dan checkpoint berikutnya terbuka. |
| 6     | Membuka Sea Passport.                                          | Sistem menampilkan progres belajar, badge, stamp, dan total bintang.                |

### Konsep QR Code

1. Satu QR Code mewakili satu pulau/checkpoint, misalnya `QR_JAWA_001` atau `QR_PAPUA_001`.
2. QR membuka halaman detail pulau yang berisi maksimal 3 biota, bukan hanya satu hewan.
3. QR dapat ditempel pada panel edukasi pulau, display aquarium, kartu misi, atau poster demo.
4. Prototype dapat menggunakan 3-5 QR aktif, sedangkan level lain dapat ditampilkan dalam status terkunci atau coming soon.
5. Jika QR gagal terbaca, pengguna dapat memasukkan kode manual sebagai alternatif akses.

### Sistem Reward

| Skor   | Hasil       | Reward                                        |
| ------ | ----------- | --------------------------------------------- |
| 90–100 | 3 bintang   | XP +15 dan stamp terbuka                      |
| 70–89  | 2 bintang   | XP +10 dan stamp terbuka                      |
| 50–69  | 1 bintang   | Perlu mengulang agar level berikutnya terbuka |
| < 50   | Belum lulus | Pengguna mengulang quiz                       |

## 8. Arsitektur Sistem dan Teknologi

| Komponen        | Rencana Implementasi Ringkas                                                                                         |
| --------------- | -------------------------------------------------------------------------------------------------------------------- |
| Mobile App      | Flutter atau React Native; halaman Home, Ocean Map, Scanner, Detail Pulau, Quiz, Result, Sea Passport, dan Progress. |
| Database        | JSON lokal atau SQLite untuk MVP. Firebase Firestore dapat digunakan untuk pengembangan online.                      |
| Auth            | Guest Mode untuk MVP. Login akun dapat dikembangkan pada tahap lanjutan.                                             |
| QR Scanner      | Library QR scanner mobile untuk membaca ID pulau, contoh `QR_JAWA_001`.                                              |
| Media           | Foto/video berlisensi, dokumentasi aquarium/museum, atau link video edukatif.                                        |
| UI/UX Tools     | Figma atau Canva untuk mockup dan poster.                                                                            |
| Diagram Alur    | draw.io atau Whimsical untuk pembuatan user flow.                                                                    |
| Version Control | GitHub untuk source code, README, instalasi, dan dokumentasi progres.                                                |

Pemilihan teknologi diarahkan pada implementasi mobile yang ringan, realistis, dan sesuai kebutuhan MVP. Sistem dapat berjalan secara lokal tanpa login penuh dan tanpa admin panel pada tahap awal pengembangan.

## 11. Kesimpulan

Oseana merupakan aplikasi edukasi biota laut Indonesia yang menggabungkan konsep pulau, QR Code, materi visual, quiz, gamifikasi, dan Sea Passport. Aplikasi ini dirancang untuk membuat pembelajaran laut menjadi lebih interaktif, menarik, dan terukur.
